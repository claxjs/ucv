> 一个可以通过组合 工具类(utility classes) 和 属性(props) 生成 可执行工具类变量 的库

## 🚀 功能

- 插槽分片
- 定制变量
- 组合变量
- 类型提示
- 不限框架

## <a name="why">❓ WHY</a>

`ucv` 主要提供一个 插槽分片 和 组合变量 功能，解决封装组件上难以控制的一些痛点

若用不到以上功能，那么我们建议直接使用 `cva` 即可
